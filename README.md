# sweet-kernel
Custom kernel untuk Xiaomi Redmi Note 10 Pro (sweet) yang dioptimalkan khusus untuk Kali   NetHunter (Monitor Mode &amp; Packet Injection) serta performa Gaming Ekstrem. Fitur: No Thermal   Throttling (Unleashed), 1000Hz Polling Rate, &amp; Support Atheros AR9271. Dibuat oleh zelocoll.
